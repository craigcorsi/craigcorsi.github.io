Online Writing Journal

This full stack web application allows users to write and update drafts of their up-and-coming creative works, chapter by chapter. User authentication keeps user drafts private, while a simple interface reduces clutter.

This is a collaboration with two other developers at the University of Minnesota Coding Boot Camp. The Github page is here.

Joke Generator Extraordinaire

Working on a new joke? It can be difficult to sift through the English lexicon to find the right word associations. This app assists with the brainstorming process by collecting a topic phrase, searching for related terms, and spitting back new and interesting phrases along with funny images.

This is a collaboration with two other developers at the University of Minnesota Coding Boot Camp. The Github page is here.
